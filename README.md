# 📊 Stock Analysis

## Overview
This project analyzes Netflix stock data using Python.

## Features
- Data cleaning and preprocessing
- Moving averages (MA50, MA200)
- RSI, MACD, Bollinger Bands
- Volatility and returns analysis
- Correlation with Apple stock

## Tools Used
- Python
- pandas
- matplotlib

## Dataset
- Netflix stock data (CSV)
- Apple stock data for comparison

## How to Run
1. Open the notebook in Google Colab
2. Upload the dataset
3. Run all cells

## Insights
- Identifies trends and momentum
- Highlights volatility patterns
- Demonstrates basic trading signals
